Guide 124

类别：气体-有毒和/或腐蚀性-氧化性

潜在危害

人身安全

  • ·有毒;吸入或经皮肤吸收可致人死亡。
  • ·火将产生刺激性、腐蚀性和/或有毒气体。
  • ·接触气体或液化气可能会导致烧伤、重伤和/或冻伤。
  • ·消防或稀释水的径流可能会造成环境污染。

燃烧或爆炸

  • ·物质不会燃烧，但会促进燃烧。
  • ·液化气中的蒸气最初比空气重，并沿地面扩散。
  • ·这些是强氧化剂，会与包括燃料在内的许多物质发生剧烈或爆炸反应
  • ·可能点燃可燃物(木材、纸张、油、衣物等)。
  • ·有些会与空气、潮湿空气和/或水发生剧烈反应。
  • ·暴露于火中的钢瓶可能通过泄压装置排出和释放有毒和/或腐蚀性气体。
  • ·容器加热时可能会爆炸。
  • ·破裂的钢瓶可能会喷射。

公共安全

公共安全

  • ·拨打911。然后拨打发货纸上的应急电话号码。如果发货纸不可用或无应答，请参阅封底内页上列出的相应电话号码。
  • ·禁止未经授权的人员进入。
  • ·保持逆风、上坡或上游。
  • ·许多气体比空气重，会沿地面扩散，并聚集在低矮或受限的区域(下水道、地下室、储罐等)。
  • ·在进入之前，对封闭空间进行通风，但前提是必须经过适当的训练和配备。

防护用具

  • ·佩戴正压自给式呼吸器(SCBA)。
  • ·当没有火灾危险时，请穿制造商特别推荐的化学防护服。
  • ·结构消防员的防护服提供热保护，但只能提供有限的化学保护。

疏散

立即采取的预防措施

  • ·将隔离溢出或泄漏区域全方位隔离至少100米(330英尺)。

泄露

  • ·参见表1-初始隔离和保护操作距离。

起火

  • ·如果发生油箱，有轨车或油罐车起火，则在各个方向进行800米（1/2英里）的隔离；另外，考虑在各个方向进行800米（1/2英里）的初始疏散。
  • ·[FLAG]在加拿大，此产品可能需要制定紧急援助计划（ERAP）。请查阅运输纸和/或ERAP程序部分（第390页）。

应急响应

起火

基本

小型火灾

CAUTION: These materials do not burn but will support combustion. Some will react violently with water.

  • ·控制火焰，让它燃烧。如果必须灭火，建议使用水雾灭火。
  • ·水;不含干粉，二氧化碳或卤化烷烃
  • ·不要在容器内取水。
  • ·如果可以安全进行，请将未损坏的容器移离火场周围区域。
  • ·损坏的钢瓶只能由专家处理。

储罐起火

  • ·从最远的距离灭火，或使用无人值守的主流设备或监控喷嘴。
  • ·用大量的水冷却容器，直到大火扑灭。
  • ·请勿将水引向泄漏源或安全装置，否则可能会结冰。
  • ·如果排气安全装置发出上升的声音或储罐变色，请立即退出。
  • ·始终远离被大火吞没的容器。
  • ·对于大火，请使用无人值守的主流设备或监控喷嘴；如果不可能，请撤离区域，让火燃烧。

溢出或泄漏

基本

  • ·请勿触摸或穿过溢出的材料。
  • ·保管易燃品(木材、纸张、油等)远离溢出的材料。
  • ·如果您可以在没有风险的情况下停止泄漏。
  • ·使用喷水减少蒸气或转移蒸气云漂移。避免让水流接触溢出的材料。
  • ·请勿将水引向溢出物或泄漏源。
  • ·如果可能，转动泄漏的容器，使气体而不是液体泄漏。
  • ·防止进入水道、下水道、地下室或受限区域。
  • ·隔离区域，直到气体扩散为止。
  • ·区域通风。

急救

基本

  • ·拨打911或紧急医疗服务。
  • ·确保医务人员了解涉及的材料并采取预防措施保护自己。
  • ·如果可以保证安全，请将患者转移到新鲜空气中。
  • ·如果患者没有呼吸，请给予人工呼吸。
  • ·如果患者摄入或吸入了该物质，请勿进行嘴对嘴复苏；在进行人工呼吸之前要洗脸和洗口。使用装有单向阀或其他适当呼吸医疗设备的袖珍口罩。
  • ·如果呼吸困难，请供给氧。
  • ·冻结在皮肤上的衣物应先解冻，然后再取下。
  • ·脱下并隔离受污染的衣服和鞋子。
  • ·如果接触到物质，立即用自来水冲洗皮肤或眼睛至少20分钟。
  • ·让受害者保持冷静并感到温暖。
  • ·保持对受害者的观察。
  • ·接触或吸入的效果可能会延迟。
