Guide 114

类别：爆炸物*-1.4或1.6项

注：*有关“兼容组”字母的信息，请参阅“词汇表”部分。

潜在危害

燃烧或爆炸

  • ·如果火灾抵达货物，可能爆炸并抛出800米（1/2英里）甚至更远的碎片。
  • ·有关“兼容性组”字母的信息，请参阅术语表部分。

人身安全

  • ·火灾可能产生刺激性、腐蚀性或有毒气体。

公共安全

公共安全

  • ·拨打911。然后拨打发货纸上的应急电话号码。如果发货纸不可用或无应答，请参阅封底内页上列出的相应电话号码。
  • ·将人移出场景视线，远离窗户。
  • ·禁止未经授权的人员进入。
  • ·保持逆风、上坡或上游。
  • ·在进入之前，对封闭空间进行通风，但前提是必须经过适当的训练和配备。

防护用具

  • ·佩戴正压自给式呼吸器(SCBA)。
  • ·结构消防员的防护服提供热保护，但只能提供有限的化学保护。

疏散

立即采取的预防措施

  • ·立即隔离所有方向上至少100米（330英尺）的溢出或泄漏区域。

大规模泄漏

  • ·考虑在各个方向进行250米（800英尺）的初始疏散。

起火

  • ·如果火车或拖车着火，请在各个方向进行800米（1/2英里）的隔离；还可以疏散包括紧急响应人员在内的所有方向的800米（1/2英里）距离。
  • ·如果火灾威胁到装有标有1.4S标签的包裹或包含1.4S材料的包裹的货物区域，请考虑在各个方向上至少隔离15米（50英尺）。
  • ·[FLAG]在加拿大，此产品可能需要制定紧急援助计划（ERAP）。请查阅运输纸和/或ERAP程序部分（第390页）。

应急响应

起火

基本

货物起火

  • ·火势达到货物时请勿灭火！货物可能会爆炸！
  • ·停止所有交通并清理沿各个方向至少800米（1/2英里）的区域燃烧。
  • ·如果货物已受热，请勿移动货物或车辆。

轮胎或车辆起火

  • ·使用大量水-充水！如果没有水，请使用二氧化碳，干粉或灰尘。
  • ·如有可能，在没有危险的情况下，使用无人值守的主控设备或在最大距离处监控喷嘴，以防止火势蔓延到货物区域。
  • ·请特别注意轮胎着火，因为这可能会引起重燃。请在安全距离内待命，并准备好灭火器以备重新点燃。

CLASS 1.4S材料起火

  • ·带有1.4S标签的包件或包含1.4S类材料的包件的设计或包装方式使得当涉及火灾时，它们可能会剧烈燃烧，并产生局部爆炸和碎片投射。
  • ·影响通常仅限于包裹附近。
  • ·在合理的距离内采取常规预防措施进行灭火。

溢出或泄漏

基本

  • ·从紧邻区域清除所有点火源(禁止吸烟、照明弹、火花或火焰)。
  • ·搬运产品时使用的所有设备必须接地。
  • ·请勿触摸或穿过溢出的材料。
  • ·请勿在100米（330英尺）的电雷管内操作无线电发射器。
  • ·除非受到专家的监督，否则请勿清理或处置。

急救

基本

  • ·拨打911或紧急医疗服务。
  • ·确保医务人员了解涉及的材料并采取预防措施保护自己。
  • ·如果可以保证安全，请将患者转移到新鲜空气中。
  • ·如果患者没有呼吸，请给予人工呼吸。
  • ·如果呼吸困难，请供给氧。
  • ·脱下并隔离受污染的衣服和鞋子。
  • ·如果接触到物质，立即用自来水冲洗皮肤或眼睛至少20分钟。
