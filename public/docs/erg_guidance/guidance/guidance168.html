Guide 168

类别：一氧化碳(冷冻液)

潜在危害

人身安全

  • ·有毒；极端危险。
  • •吸入极其危险的;可能是致命的。
  • ·接触气体或液化气可能会导致烧伤、重伤和/或冻伤。
  • •无气味，不会被嗅觉察觉。

燃烧或爆炸

  • ·极易燃。
  • •注意:火焰可以是隐形的。使用另一种检测方法(热相机，扫帚柄等)
  • ·可能被热量、火花或火焰点燃。
  • ·容器加热时可能会爆炸。
  • ·室内、室外或下水道有蒸汽爆炸和中毒危险。
  • ·液化气中的蒸气最初比空气重，并沿地面扩散。
  • ·蒸汽可能会流向点火源并闪回。
  • ·径流可能会造成火灾或爆炸危险。

公共安全

公共安全

  • ·拨打911。然后拨打发货纸上的应急电话号码。如果发货纸不可用或无应答，请参阅封底内页上列出的相应电话号码。
  • ·禁止未经授权的人员进入。
  • ·保持逆风、上坡或上游。
  • ·许多气体比空气重，会沿地面扩散，并聚集在低矮或受限的区域(下水道、地下室、储罐等)。
  • ·在进入之前，对封闭空间进行通风，但前提是必须经过适当的训练和配备。

防护用具

  • ·佩戴正压自给式呼吸器(SCBA)。
  • ·当没有火灾危险时，请穿制造商特别推荐的化学防护服。
  • ·结构消防员的防护服提供热保护，但只能提供有限的化学保护。
  • ·在处理冷藏/低温液体时，请始终穿上防热衣物。

疏散

立即采取的预防措施

  • ·将隔离溢出或泄漏区域全方位隔离至少100米(330英尺)。

泄露

  • ·参见表1-初始隔离和保护操作距离。

起火

  • ·如果发生油箱，有轨车或油罐车起火，则在各个方向进行800米（1/2英里）的隔离；另外，考虑在各个方向进行800米（1/2英里）的初始疏散。

应急响应

起火

基本

CAUTION: Flame can be invisible. Use an alternate method of detection (thermal camera, broom handle, etc.)

  • ·除非可以阻止泄漏，否则请不要灭火。

小型火灾

  • 干燥化学品，二氧化碳或水喷雾。

大型火灾

  • ·喷水、雾或普通泡沫。
  • ·如果可以安全进行，请将未损坏的容器移离火场周围区域。

储罐起火

  • ·从最远的距离灭火，或使用无人值守的主流设备或监控喷嘴。
  • ·用大量的水冷却容器，直到大火扑灭。
  • ·请勿将水引向泄漏源或安全装置，否则可能会结冰。
  • ·如果排气安全装置发出上升的声音或储罐变色，请立即退出。
  • ·始终远离被大火吞没的容器。

溢出或泄漏

基本

  • ·从紧邻区域清除所有点火源(禁止吸烟、照明弹、火花或火焰)。
  • ·搬运产品时使用的所有设备必须接地。
  • ·请勿触摸或穿过溢出的材料。
  • ·如果您可以在没有风险的情况下停止泄漏。
  • ·使用喷水减少蒸气或转移蒸气云漂移。避免让水流接触溢出的材料。
  • ·请勿将水引向溢出物或泄漏源。
  • ·如果可能，转动泄漏的容器，使气体而不是液体泄漏。
  • ·防止进入水道、下水道、地下室或受限区域。
  • ·隔离区域，直到气体扩散为止。

急救

基本

  • ·拨打911或紧急医疗服务。
  • ·确保医务人员了解涉及的材料并采取预防措施保护自己。
  • ·如果可以保证安全，请将患者转移到新鲜空气中。
  • ·如果患者没有呼吸，请给予人工呼吸。
  • ·如果呼吸困难，请供给氧。
  • ·脱下并隔离受污染的衣服和鞋子。
  • ·如果接触到物质，立即用自来水冲洗皮肤或眼睛至少20分钟。
  • ·如果接触到液化气，请用温水解冻磨砂部件。
  • ·让受害者保持冷静并感到温暖。
  • ·保持对受害者的观察。
  • ·接触或吸入的效果可能会延迟。
