Guide 111

类别：混合负载/不明货物

潜在危害

燃烧或爆炸

  • ·可能因热、冲击、摩擦或污染而爆炸。
  • ·与空气、水或泡沫接触时可能会发生剧烈或爆炸反应。
  • ·可能被热量、火花或火焰点燃。
  • ·蒸汽可能会流向点火源并闪回。
  • ·容器加热时可能会爆炸。
  • ·破裂的钢瓶可能会喷射。

人身安全

  • ·吸入、摄入或接触物质可能导致严重伤害、感染、疾病或死亡。
  • ·高浓度气体可能会在没有警告的情况下导致窒息。
  • ·接触可能会导致皮肤和眼睛烧伤。
  • ·与火或水接触可能会产生刺激性、有毒或腐蚀性气体。
  • ·消防或稀释水的径流可能会造成环境污染。

公共安全

公共安全

  • ·拨打911。然后拨打发货纸上的应急电话号码。如果发货纸不可用或无应答，请参阅封底内页上列出的相应电话号码。
  • ·禁止未经授权的人员进入。
  • ·保持逆风、上坡或上游。

防护用具

  • ·佩戴正压自给式呼吸器(SCBA)。
  • ·结构消防员的防护服提供热保护，但只能提供有限的化学保护。

疏散

立即采取的预防措施

  • ·将隔离溢出或泄漏区域全方位隔离至少100米(330英尺)。

起火

  • ·如果发生油箱，有轨车或油罐车起火，则在各个方向进行800米（1/2英里）的隔离；另外，考虑在各个方向进行800米（1/2英里）的初始疏散。

应急响应

起火

基本

CAUTION: Material may react with extinguishing agent.

小型火灾

  • ·干粉，二氧化碳，喷水或普通泡沫。

大型火灾

  • ·喷水、雾或普通泡沫。
  • ·如果可以安全进行，请将未损坏的容器移离火场周围区域。

储罐起火

  • ·用大量的水冷却容器，直到大火扑灭。
  • ·不要在容器内取水。
  • ·如果排气安全装置发出上升的声音或储罐变色，请立即退出。
  • ·始终远离被大火吞没的容器。

溢出或泄漏

基本

  • ·请勿触摸或穿过溢出的材料。
  • ·从紧邻区域清除所有点火源(禁止吸烟、照明弹、火花或火焰)。
  • ·搬运产品时使用的所有设备必须接地。
  • ·保管易燃品(木材、纸张、油等)远离溢出的材料。
  • ·使用喷水减少蒸气或转移蒸气云漂移。避免让水流接触溢出的材料。
  • ·防止进入水道、下水道、地下室或受限区域。

小规模泄露

  • ·用沙子或其他不燃吸收性材料吸收并放入容器中，以备后用。

大规模泄漏

  • ·在液体溢出之前筑堤以备后用。

急救

基本

  • ·拨打911或紧急医疗服务。
  • ·确保医务人员了解涉及的材料并采取预防措施保护自己。
  • ·如果可以保证安全，请将患者转移到新鲜空气中。
  • ·如果患者没有呼吸，请给予人工呼吸。
  • ·如果患者摄入或吸入了该物质，请勿进行嘴对嘴复苏；在进行人工呼吸之前要洗脸和洗口。使用装有单向阀或其他适当呼吸医疗设备的袖珍口罩。
  • ·如果呼吸困难，请供给氧。
  • ·脱下并隔离受污染的衣服和鞋子。
  • ·如果接触到物质，立即用自来水冲洗皮肤或眼睛至少20分钟。
  • ·淋浴并用肥皂和水清洗。
  • ·让受害者保持冷静并感到温暖。
  • ·接触物质(吸入、摄入或皮肤接触)的影响可能会延迟。
