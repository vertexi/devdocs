Guide 173

类别：吸附气体-有毒*

注：*一些物质也可能是易燃、腐蚀性和/或氧化

潜在危害

人身安全

  • ·有毒;吸入或经皮肤吸收可致人死亡。
  • •蒸汽可能是刺激性的。
  • •与气体接触可能导致烧伤和伤害。
  • ·火将产生刺激性、腐蚀性和/或有毒气体。
  • ·消防或稀释水的径流可能会造成环境污染。

燃烧或爆炸

  • •有些气体可能燃烧或被热、火花或火焰点燃。
  • ·可能与空气形成爆炸性混合物。
  • •氧化剂可以点燃可燃物(木材、纸张、石油、衣物等)，但由于运输压力小，不易燃烧。
  • ·蒸汽可能会流向点火源并闪回。
  • ·其中一些物质可能与水剧烈反应。
  • ·暴露在火灾中的钢瓶可能会通过泄压装置排出有毒和易燃气体。
  • •径流可能造成火灾危险。

公共安全

公共安全

  • ·拨打911。然后拨打发货纸上的应急电话号码。如果发货纸不可用或无应答，请参阅封底内页上列出的相应电话号码。
  • ·禁止未经授权的人员进入。
  • ·保持逆风、上坡或上游。
  • ·许多气体比空气重，会沿地面扩散，并聚集在低矮或受限的区域(下水道、地下室、储罐等)。
  • ·在进入之前，对封闭空间进行通风，但前提是必须经过适当的训练和配备。

防护用具

  • ·佩戴正压自给式呼吸器(SCBA)。
  • ·当没有火灾危险时，请穿制造商特别推荐的化学防护服。
  • ·结构消防员的防护服提供热保护，但只能提供有限的化学保护。

疏散

立即采取的预防措施

  • ·将隔离溢出或泄漏区域全方位隔离至少100米(330英尺)。

泄露

  • ·参见表1-初始隔离和保护操作距离。

起火

  • •如果几个小包裹(在火车车厢或拖车里)卷入火灾，在1600米(1英里)的各个方向隔离;同时，考虑从各个方向疏散1600米(1英里)。
  • ·[FLAG]在加拿大，此产品可能需要制定紧急援助计划（ERAP）。请查阅运输纸和/或ERAP程序部分（第390页）。

应急响应

起火

基本

  • ·除非可以阻止泄漏，否则请不要灭火。

小型火灾

  • ·干化学、CO2、水雾或抗酒精泡沫。
  • •UN3515, UN3518, UN3520只使用水;不含干粉，二氧化碳或哈龙。

大型火灾

  • ·水雾、防酒精泡沫。
  • ·不要在容器内取水。
  • ·如果可以安全进行，请将未损坏的容器移离火场周围区域。
  • ·损坏的钢瓶只能由专家处理。

涉及几个小包裹的火灾（在轨道车或拖车内）

  • ·从最远的距离灭火，或使用无人值守的主流设备或监控喷嘴。
  • ·用大量的水冷却容器，直到大火扑灭。
  • •不要将水直接注入泄漏源或安全装置。
  • ·如果排气安全装置发出上升的声音或储罐变色，请立即退出。
  • ·始终远离被大火吞没的容器。

溢出或泄漏

基本

  • •有些气体可能是易燃的。消除附近所有火源(禁止吸烟、信号弹、火花或火焰)。
  • •对于易燃气体，在处理产品时使用的所有设备必须接地。
  • •对于氧化性物质，应远离易燃物质(木材、纸张、油等)。
  • ·请勿触摸或穿过溢出的材料。
  • ·如果您可以在没有风险的情况下停止泄漏。
  • ·请勿将水引向溢出物或泄漏源。
  • ·使用喷水减少蒸气或转移蒸气云漂移。避免让水流接触溢出的材料。
  • ·防止进入水道、下水道、地下室或受限区域。
  • ·隔离区域，直到气体扩散为止。

急救

基本

  • ·拨打911或紧急医疗服务。
  • ·确保医务人员了解涉及的材料并采取预防措施保护自己。
  • ·如果可以保证安全，请将患者转移到新鲜空气中。
  • ·如果患者没有呼吸，请给予人工呼吸。
  • ·如果患者摄入或吸入了该物质，请勿进行嘴对嘴复苏；在进行人工呼吸之前要洗脸和洗口。使用装有单向阀或其他适当呼吸医疗设备的袖珍口罩。
  • ·如果呼吸困难，请供给氧。
  • ·脱下并隔离受污染的衣服和鞋子。
  • ·如果接触到物质，立即用自来水冲洗皮肤或眼睛至少20分钟。
  • ·万一发生灼伤，请立即用冷水尽可能长时间地冷却受影响的皮肤。如果粘附在皮肤上，请勿脱下衣服。
  • ·让受害者保持冷静并感到温暖。
  • ·保持对受害者的观察。
  • ·接触或吸入的效果可能会延迟。
